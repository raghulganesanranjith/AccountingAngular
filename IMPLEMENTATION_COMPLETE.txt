/* HOW TO USE */

1. Navigate to Sales Quotes
2. Look at the table - you'll see demo quotes mixed with regular ones
3. Demo quotes have:
   - DEMO badge next to the quote number (pink gradient)
   - Gradient left border to highlight
   - "Click to edit" cursor on hover
4. Click any demo quote row
5. Form opens with ALL fields automatically filled
6. Edit any field
7. Click Save to update
8. Changes persist in localStorage

/* KEY FEATURES */

✅ Demo records in the SAME table (not separate)
✅ One click to open form
✅ Auto-filled fields (no manual entry needed)
✅ Persistent changes (localStorage)
✅ Real-time sync across components
✅ Works for all modules (Sales, Purchase, Common, etc.)

/* IMPLEMENTATION FOR OTHER MODULES */

Copy the pattern from Sales Quotes:

1. Create demo records array in initializeDemoRecords()
2. Register: this.demoDataManager.registerDemoRecords(type, module, records)
3. In table row add:
   - [class.demo-row]="record.id.includes('demo')"
   - (click)="openRecord(record)"
   - class="clickable-row"
4. Add DEMO badge: <span *ngIf="record.id.includes('demo')" class="demo-badge-inline">DEMO</span>
5. Implement openRecord() to fill form
6. Add CSS for .demo-row, .clickable-row, .demo-badge-inline

See DEMO_DATA_IMPLEMENTATION_GUIDE.ts for detailed example code.
